# vinscully
A Raspberry Pi / Python app that will play Vin Scully's "It's time for Dodger Baseball" at the start of every Dodger game.

# Get Started

1. Install pip
  
  ```
  sudo apt-get install python-pip
  ````

2. Clone repository
  
  ```
  cd ~/
  git clone https://github.com/Lukeas14/vinscully.git
  ```
